# Social Network
### Running the Application

1. **Clone the repository**
   ```bash
   git clone git@github.com:goIdioms/social-network.git
   cd social-network
   ```

2. **Start the backend**
   ```bash
   docker-compose up --build
   ```

3. **Access the API**
   - Backend: http://localhost:3000
   - Health check: http://localhost:3000/hello
